# Aptos Documentation

This folder contains comprehensive documentation for building on Aptos with Privy.

## Documentation Files

- **[APTOS_CODE_SNIPPETS.md](./APTOS_CODE_SNIPPETS.md)** - Copy-paste ready code snippets for Aptos integration
- **[APTOS_DEMO_FEATURES.md](./APTOS_DEMO_FEATURES.md)** - Complete list of features and functions available in the demo
- **[APTOS_FUNCTIONS_EXAMPLES.md](./APTOS_FUNCTIONS_EXAMPLES.md)** - Detailed examples of all Aptos functions

## Quick Start

1. Check out [APTOS_CODE_SNIPPETS.md](./APTOS_CODE_SNIPPETS.md) for ready-to-use code snippets
2. Review [APTOS_DEMO_FEATURES.md](./APTOS_DEMO_FEATURES.md) to understand available features
3. Use [APTOS_FUNCTIONS_EXAMPLES.md](./APTOS_FUNCTIONS_EXAMPLES.md) for detailed function documentation

## Usage

All code snippets are production-ready and can be copied directly into your project. They include:
- Full TypeScript typing
- Error handling
- Both Privy and generic implementations
- Complete React components
- Utility functions
